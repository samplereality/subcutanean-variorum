Subcutanean Variorum

                
            

            

                
Overview

                
Every copy of Aaron Reed‚Äôs novel Subcutanean is different. Using a combinatorial narrative system, Reed generated thousands of unique versions by varying word choices, sentence structures, and entire passages.
 
                
The Subcutanean Variorum allows readers to compare any two copies side-by-side to discover how the story shifts between versions.


                
Features

                
    
                    
  • 25 pre-loaded unique versions (seeds 45443-45467)
    •  
                    
  • Generate new versions and upload them for comparison
    • 
                    
  • Multiple view modes, including Side-by-side, Track Changes, Collation, and Source Code
    • 
                    
  • Bookmark your views, keyword search, word differential analysis, and more
    • 
                


                
How to Use

                
    
                    
  1. Select two versions using the dropdown menus
    2. 
                    
  2. Choose a view mode (Unified, Side-by-side, Track Changes, or Collation)
    3. 
                    
  3. Navigate chapters using the chapter menu
    4. 
                    
  4. Search for text or analyze word differences
    5. 
                    
  5. Generate your own unique copy of the novel
    6. 
                    
  6. Upload additional EPUB or TXT copies of Subcutanean; Aaron Reed has made 10,000 variants readily available for download.
    7. 
                


                
Credits

                
In 2025 Reed released the source text and code of Subcutanean under a CC-BY 4.0 license, meaning anyone can copy, share, modify, or remix the novel as long as credit for the original is given to Reed.
 
                    
                
The Subcutanean Variorum is designed by  Mark Sample. The project code is available on GitHub.
